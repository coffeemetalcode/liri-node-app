# liri-node-app

A command line app for finding answers to your questions.

Search for:
* Upcoming concerts
* Songs from Spotify
* Movies by title
* A random concert, Spotify, or movie title search based on text contained in a local file.

*Usage*
node liri [arg 1, arg 2]

where 'arg 1' is one of the following:

* concert-this
* spotify-this-song
* movie-this
* do-what-it-says
